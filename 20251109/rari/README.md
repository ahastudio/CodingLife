# Rari Demo Application

## Install dependencies

npm으로 설치하면 `rari` 바이너리가 설치되지 않는 문제가 있습니다.

```bash
pnpm install
```

## Lint and fix

```bash
pnpm run lint
```

## Type check

```bash
pnpm run typecheck
```

## Start development server

```bash
pnpm run dev
```

Visit <http://localhost:5173> to see your app.

## Build for production

```bash
pnpm run build
```

## Preview production build

```bash
pnpm start
```
