# CodeceptJS 예제

## 설치

```bash
npm init -y

npx create-codeceptjs .
```

```bash
npm uninstall @codeceptjs/examples
```

```bash
npx playwright install
```

```bash
npx codeceptjs init
```

## 테스트 실행

```bash
npm run codeceptjs
```

```bash
npm run codeceptjs:headless
```

## 코드 정리

```bash
npm run lint
```

## 코드 검사

```bash
npm run check
```
