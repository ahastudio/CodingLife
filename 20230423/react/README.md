# Shop Demo

Install dependencies:

```bash
npm ci

npx playwright install
```

Run tests:

```bash
npm test
```

Run dev server:

```bash
npm start
```

Generate static files:

```bash
npm run build
```

Serve static files:

```bash
npm run serve
```

Run E2E test:

```bash
npm run codeceptjs

# headless mode
npm run codeceptjs:headless
```
