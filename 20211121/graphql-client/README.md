# GraphQL Client

- <https://github.com/FormidableLabs/urql>

```bash
npm init -y

npm i -D typescript eslint

npx eslint --init

npx tsc --init

npm i react react-dom
npm i -D @types/react @types/react-dom

npm i urql graphql
npm i -D @types/graphql

npm i -D parcel
```

```bash
npm start
```

<http://localhost:8080/>
