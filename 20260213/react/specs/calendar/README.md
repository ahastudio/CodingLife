# 달력

## Background

현재 프로젝트에 날짜를 시각적으로 확인할 수 있는 달력 기능이 없다.
사용자가 오늘 날짜를 빠르게 파악하고, 월 단위로 날짜를 탐색할 수 있는
간단한 달력 UI가 필요하다.

## Goal

오늘 날짜를 기준으로 월별 달력을 표시하고, 이전/다음 달로 이동하며,
언제든 오늘 날짜로 돌아올 수 있는 달력 애플리케이션을 제공한다.

## How it works

### 달력 표시

- 오늘 날짜 기준으로 해당 월의 달력을 그리드 형태로 표시한다.
- 달력 위에 현재 보고 있는 연도와 월을 표시한다.
- 오늘 날짜는 시각적으로 눈에 띄게 강조 표시한다.
- 일요일부터 토요일까지 요일 헤더를 표시한다.

### 월 이동

- 연월 표시 왼쪽에 이전 달 버튼을 배치하여 누르면 이전 달로 이동한다.
- 연월 표시 오른쪽에 다음 달 버튼을 배치하여 누르면 다음 달로 이동한다.

### 오늘로 돌아오기

- "오늘" 버튼을 항상 표시하여, 누르면 오늘 날짜가 포함된 월로 돌아온다.
